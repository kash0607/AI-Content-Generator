# ✍️ AI Content Generator

## Overview

AI Content Generator is a Streamlit-based application developed as part of the WeIntern Artificial Intelligence Internship Week 2 Assignment.

The application uses Google's Gemini AI model to generate high-quality content based on user-provided topics and tone preferences.

Users can generate blog posts and social media captions instantly with the help of Generative AI.

---

## Features

* AI-powered content generation
* Blog post generation
* Social media caption generation
* Multiple tone options
* Content regeneration
* Word count calculation
* Character count calculation
* Interactive Streamlit UI
* Gemini AI integration

---

## Technologies Used

* Python 3

## Installation

### Clone Repository

```bash
git clone <repository-url>
```

### Create Virtual Environment

```bash
python -m venv .venv
```

### Activate Environment

```bash
.venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Running the Application

```bash
streamlit run app.py
```

or

```bash
py -m streamlit run app.py
```

---

## How It Works

1. Enter a topic.
2. Select content type.
3. Choose a tone.
4. Click Generate Content.
5. Gemini AI generates content.
6. View word count and character count.
7. Use Regenerate to create a new version.
